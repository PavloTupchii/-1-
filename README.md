## Варіант завдання

Розрахунок параметрів виконується на основі номера залікової книжки (№ = 5120):

1. **C5 (Операція):** 5120 % 5 => остача 0
   * Дія: C = a * B, де a — константа.
2. **C7 (Тип даних):** 5120 % 7 => остача 3
   * Тип елементів матриці: `int`.
3. **C11 (Дія з матрицею):** 5120 % 11 => остача 5
   * Завдання: Обчислити суму найбільших елементів у рядках матриці з непарними номерами та найменших елементів у рядках матриці з парними номерами.

## Опис реалізації

Програма виконує наступні кроки:

1. Ініціалізує та заповнює випадковими цілочисельними значеннями (`int`) матрицю та задає константу для множення.
2. Обчислює нову матрицю шляхом множення початкової матриці на задану константу.
3. Проходить по кожному рядку нової матриці для розрахунку:
   – найбільших елементів у рядках з непарними номерами;
   – найменших елементів у рядках з парними номерами.
4. Обчислює їх загальну суму та виводить усі результати (матриці та фінальну суму) у консоль.
