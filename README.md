## 🚀 Як запустити
1. Склонувати репозиторій.
2. Відкрити як Maven-проект в IntelliJ IDEA.
3. Запустити `src/main/java/org/example/Main.java`.

## 🧪 Тестування
Проект містить JUnit 5 тести. Для запуску виконайте:
`mvn test` або запустіть `MainTest.java` в IDE.
